## Guia de Pistas de Skate da Zona Leste: Uma Aventura Sobre Rodinhas 

###  Sobre o Projeto

Este projeto web te ajuda a encontrar a pista de skate perfeita na Zona Leste! ✨ Com poucos cliques, você encontra informações detalhadas sobre diversas pistas, como localização, descrição e até mesmo avaliações. 

**Funcionalidades:**

* **Busca Rápida:** Encontre a pista ideal com apenas algumas palavras-chave.
* **Informações Detalhadas:** Descubra tudo sobre cada pista: localização, descrição e links para mais informações.
* **Interface Intuitiva:** Navegue facilmente pelo site e encontre as informações que você precisa.

### ️ Tecnologias Utilizadas

* **HTML:** A estrutura básica da página.
* **CSS:** Estiliza a página, deixando-a com uma cara incrível.
* **JavaScript:** Adiciona interatividade, como a função de busca.

### ⚙️ Como Funciona

1. **Busca:** Digite o nome da pista, uma palavra-chave ou qualquer outra informação relevante no campo de pesquisa.
2. **Resultados:** A mágica acontece! A aplicação busca em um banco de dados (no arquivo `dados.js`) e exibe os resultados que correspondem à sua pesquisa.
3. **Detalhes:** Clique em uma pista para ver mais informações e um link para o Google Maps.

###  Como Usar

1. **Clone o repositório:** Faça um clone deste repositório para sua máquina local.
2. **Abra o arquivo `index.html`:** Utilize um navegador web para abrir o arquivo `index.html`.
3. **Comece a explorar:** Divirta-se buscando pelas pistas de skate mais legais da Zona Leste!

###  Contribuições
